
Приложение на tkinter

Используем модель проектировани MVC
Способ хранения данных OOP

Используемые виджеты tkinter:

- Frame
- LabelFrame
- Radiobutton
- Label
- Button
- Entry

В данной реализации можно обойтись более коротким кодом,
путем создания одного шаблона окна(в данной реализации для переключения ипользуются два окна WinFirst, WinSecond)
Это делает код менее DRY. Но это сделанно намеренно, в качестве тренировки!!!
